# 🚀 موقع المطري OTP

موقع متكامل لجلب أرقام وهمية واستقبال أكواد التحقق من قنوات تيليجرام.

## ✨ المميزات

- 🎨 **تصميم عصري** - خلفية Matrix Digital Rain متحركة
- 📰 **شريط أخبار** - متحرك في أسفل الصفحة
- 📱 **متجاوب** - يعمل على جميع الأجهزة
- ⚙️ **لوحة تحكم شاملة** - تحكم كامل في الموقع
- 🔐 **حماية متقدمة** - نظام حماية بلوحة التحكم
- 📊 **إحصائيات** - متابعة شاملة للنشاط
- 💾 **نسخ احتياطي** - حفظ واستعادة الإعدادات

## 🚀 التشغيل

```bash
# تثبيت المتطلبات
pip install -r requirements.txt

# تشغيل الموقع
python app.py

# أو مع تحديد البورت
PORT=8080 python app.py
```

## 📂 هيكل الملفات

```
/workspace/
├── app.py                 # التطبيق الرئيسي

├── requirements.txt       # المتطلبات
├── README.md              # هذا الملف
└── templates/
    ├── index.html         # الصفحة الرئيسية
    ├── admin.html         # لوحة التحكم
    └── admin_login.html   # صفحة تسجيل الدخول
```

## 🔧 لوحة التحكم

- **الرابط**: `/admin`
- **كلمة السر**: افتراضياً غير مطلوبة (يمكنك إضافتها من لوحة التحكم)

### أقسام لوحة التحكم:

1. **📊 الرئيسية** - إحصائيات وسريعة
2. **📝 مدير النصوص** - تعديل نصوص الموقع
3. **🔗 مدير الروابط** - روابط التواصل
4. **📱 المنصات والدول** - إدارة الكومبوهات
5. **🔑 الأكواد المسحوبة** - عرض وحذف الأكواد
6. **📢 الإعلانات** - إدارة الإعلانات
7. **🎨 إعدادات المظهر** - الألوان والتأثيرات
8. **👥 إدارة المستخدمين** - عرض وحظر المستخدمين
9. **💾 النسخ الاحتياطي** - حفظ واستعادة
10. **🔒 الأمان** - تغيير كلمة السر
